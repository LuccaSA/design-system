Buttons

Sesame snaps gummi bears candy canes macaroon sweet roll. Danish biscuit icing marshmallow candy cotton candy powder.

Basic usage

<button class="button">...</button> <button class="button mod-flat">...</button> <button class="button mod-wired">...</button> <button class="button mod-filling">...</button> Tip: Can be used as <a href="#">, for links.
Tip: You can use mod-link, to give a link aspect to your button.

Palette

<button class="button">...</button> <button class="button palette-primary">...</button> <button class="button palette-secondary">...</button> <button class="button palette-success">...</button> <button class="button palette-warning">...</button> <button class="button palette-error">...</button>

Size

<button class="button size-smaller">...</button> <button class="button size-small">...</button> <button class="button size-big">...</button> <button class="button size-bigger">...</button>

States

<button class="button is-loading">...</button> <button class="button is-success">...</button> <button class="button is-error">...</button>

Disabled

<button class="button" disabled>...</button> Tip: Buttons can also be disabled with a class class="button is-disabled", but you will not be able to use a tooltip on it.
