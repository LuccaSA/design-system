Textfields

Basique

<div class="textfield"> <input class="textfield-input" id="textfieldID" type="text" placeholder="placeholder"> <label class="textfield-label" for="textfieldID">...</label> </div> info Utilisez <div class="textfield mod-block"> pour un textfield prenant toute la largeur
info Utilisez <div class="textfield mod-inline"> pour passer le textfield en inline-block

Apparence

Compact

<div class="textfield mod-compact"> <input class="textfield-input" type="text"> <label class="textfield-label">...</label> </div>

Material

<div class="textfield mod-material"> <input class="textfield-input" type="text"> <label class="textfield-label">...</label> </div>
info Utilisez <div class="textfield-input is-filled"> pour garder le label en position haute en mode material

Largeur

Utilisez un des mods suivants pour changer la largeur du textfield :
mod-shortest,mod-shorter,mod-short, mod-long,mod-longer or mod-longest

Disabled

<div class="textfield"> <input class="textfield-input" id="textfieldID" type="text" disabled> <label class="textfield-label" for="textfieldID">...</label> </div>

Helper / Warning / Error

Few tips about the input
<div class="textfield"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label" for="textfieldID">...</label> <div class="textfield-messages"> <div class="textfield-messages-helper">...</div> </div> </div>
Humm, there is a warning...
<div class="textfield"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label" for="textfieldID">...</label> <div class="textfield-messages"> <div class="textfield-messages-warning">...</div> </div> </div>
Oops, there is an error!
<div class="textfield"> <input class="textfield-input is-error" id="textfieldID" type="text"> <label class="textfield-label" for="textfieldID">...</label> <div class="textfield-messages"> <div class="textfield-messages-error">...</div> </div> </div>

Validation

You can add a is-loading and a is-valid or is-invalid class on <div class="textfield"> for your inline validation.

Brancher les interactions valid / invalid

Suffix

€
<div class="textfield mod-withSuffix"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label" for="textfieldID">Label textfield</label> <div class="textfield-suffix">€</div> </div> info Pour utiliser un suffix sans textfield-label, il est nécessaire d'ajouter une classe mod-noLabel au textfield

Recherche

<div class="textfield mod-search"> <input class="textfield-input" id="textfieldID" type="text" placeholder="e.g. Clark Kent"> <label class="textfield-label" for="textfieldID">Search</label> </div>

Requis

<div class="textfield is-required"> <input class="textfield-input" id="textfieldID" type="text"> <label class="textfield-label" for="textfieldID">...</label> </div>

Multi-lignes

<div class="textfield mod-multiline"> <textarea class="textfield-input" id="textfieldID"></textarea> <label class="textfield-label" for="textfieldID">Textarea label</label> </div>

Radio

Le mod "radio" vous permet d'afficher un textfield à l'interieur d'un label de radio.

<div class="radio"> <input class="radio-input" type="radio" id="radioID"> <label class="radio-label" for="radioID"> Or <div class="textfield mod-radio"> <input class="textfield-input" type="text"> </div> </label> </div>
